# BlockchainStand

## Что представляет собой данный проект?
Данный проект является реализацией лабораторного стенда по технологии Blockchain.

Что используется в этом стенде:
    1. Docker
    2. Python
    3. Docker-образ `ethereum/client-go:v1.13.15`


Данный стенд предназначен для знакомства с технологией Blockchain на примере Ethereum.
Он позволяет запустить приватную Ethereum-сеть c выбором алгоритма консенсуса. На данный момент
можно выбрать только алгоритм Clique (PoA).
